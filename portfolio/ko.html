고려개발(주)

2018.08.27

고려개발(주) 사이트는 기업 사이트로서 깔끔한 디자인과 색상을 사용하여 제작한 사이트로 마크업과 css기능을 100% 발휘한 사이트로 다양한 jQuery 기능과 플러그인을 사용하여 제작하였기에 선택하여 포토폴리오 제작에 적합하다 판단하여 제작하였습니다.

메인 페이지

고려개발 사이트의 메인 부분으로서 header / section / footer의 구조로 제작하였고 section태그 안 첫번째 슬라이드, 세번째 슬라이드는 bxslide를 이용하여 슬라이드 이미지를 제작하였으며 두번째 슬라이드는 직접 코딩하여 제작하였습니다. 두번째 슬라이드는 플러그인을 사용하기 보다는 위치값을 구하여 슬라이드 되는것이기 때문에 직접 코딩하여 제작하였습니다

주요 서브 페이지

가치체계

가치체계라는 페이지를 특별하게 보이기 위해 새로운 창으로 열리게 하였고 fullpage플러그인을 사용하여 좀 더 쉽고 간편하게 볼 수 있도록 제작하였습니다.

주요 서브 페이지

주요 연혁

각 년도별 연혁을 나타내기 위해 슬라이드 이미지/버튼으로 구성하였으며 각 버튼에 맞는 년도의 컨텐츠로 이동 할 수 있게 코딩하였습니다.

주요 서브 페이지

찾아오시는 길

네이버 지도 API에 등록후 코딩하여 고려개발(주) 기업의 위치를 지도 안에 나타내었습니다.

주요 서브 페이지

사업 소개

여러가지 사업 이미지를 표시하고 동적으로 배치하기 위해 mansonry플러그인을 사용하여 좀더 역동적이고 화려하게 제작하였습니다.

주요 서브 페이지

역사관

역사관이라는 페이지는 고려개발 사이트안 특별한 사이트로 기존의 구조와 달리 fullpage 플러그인을 사용하여 특별하게 제작하였고 페이지안 슬라이드 이미지는 직접 코딩하여 제작하였습니다.

주요 서브 페이지

직무 소개

각 사원을 소개하는 페이지로 사원들의 이미지를 나열 후 클릭시 사원들의 인터뷰 내용을 facybox 플러그인을 사용하여 모달창을 띄움으로써 강조 하여 제작하였습니다.

가치체계 주요연혁 찾아오시는 길 사업소개 역사관 직무소개

에필로그

주요 연혁 페이지와 역사관 페이지의 슬라이드 이미지를 직접 코딩하다 보니 많이 힘들었지만 해결을 되었을 때 너무나도 행복하고 뿌듯했습니다. 처음에는 많은 양의 페이지를 제작해야 하다 보니 겁을 먹긴 하였지만 하나하나 완성돼가는 모습을 보면서 이래서 웹 사이트를 제작하는구나라는 뿌듯함을 느끼고 행복함에 지난 힘들게 만들었던 날들이 하나도 힘들지 않았고 오히려 다음 다른 사이트 제작에 더 잘 할 수 있겠다는 자신감을 얻을 수 있는 포트폴리오 제작이었습니다.
