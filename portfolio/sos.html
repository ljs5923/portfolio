SOSFESTIVAL

2018.09.21

SOSFESTIVAL 사이트는 서울문화로 바캉스 안내 페이지로 광장 바캉스, 아트 바캉스, 커뮤니티, 영어 안내 페이지로 나누어져 있으며 시원한 색상과 한눈에 들어오는 디자인과 다양한 동적 기능이 있어 포트폴리오 제작에 적합하다 생각하여 제작하게 되었습니다.

메인 페이지

SOSFESTIVAL 사이트의 메인 페이지로써 기존의 스크롤을 계속하면서 봐야 하는 불편한 점을 해소하고자 fullpage.js 플러그인을 사용하여 구역을 나누어 좀 더 보기 쉽게 구현하였으며 기존의 복잡하던 이미지들을 깔끔하게 정렬하여 사이트 방문자가 알아보기 쉽게 제작하였습니다.

주요 서브 페이지

서울광장바캉스

기존의 있던 사이드 콘텐츠 정보가 사이트 내에 많이 중복되어 필요가 없음을 느껴고 제거 후 좀 더 깔깔 끔하게 보이기 위해 콘텐츠들을 위치 시켰습니다.

주요 서브 페이지

청계광장바캉스

기존의 있던 사이드 콘텐츠 정보가 사이트 내에 많이 중복되어 필요가 없음을 느껴고 제거 후 좀 더 깔깔 끔하게 보이기 위해 콘텐츠들을 위치 시켰습니다.

광화문광장바캉스

주요 서브 페이지

MODAL

서울광장 바캉스 / 청계광장 바캉스의 주요 안내 내용을 해당 콘텐츠를 클릭하면 모 들창이 뜨도록 제작하였습니다. fancybox.js 플러그인을 이용하여 각 해당하는 컨텐트에 맞는 안내 내용을 구현하였습니다.
